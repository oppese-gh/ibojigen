# ぼじろーぐ (Bojirogue)

ダンジョンの奥深くを目指せ…。

シンプル操作で遊べる、  
絵文字ローグライクダンジョン探索ゲーム。

プレイヤーはダンジョンを降りながら、
モンスターと戦い、アイテムを拾い、成長していきます。  
目指すは最深部―― **30階。**

しかしその先には、  
このダンジョンの主が待っています。

---

## 🎮 Game Features

**シンプル操作 × ローグライク**

- ランダム生成ダンジョン
- 階層ごとに強くなるモンスター
- 食料・武器・装備・特殊アイテム
- 戦略が変わる満腹度システム
- 突然訪れるモンスターハウス
- レベルアップによる成長
- ハイスコア記録

運が良ければ、  
とてもありがたい存在に出会えるかもしれません。

## 🔵 Ibo System（いぼシステム）

このゲームには **「イボ」** と呼ばれる特殊なエネルギーがあります。

イボは主に2つの使い方があり、１度使うと残数が１減ります。

モンスターを倒したり探索を続けることで  
イボが増えることがあります。
プレイヤーのレベルが上がることで、イボの回復がわずかに早くなります。

### ✨ イボビーム（Aキー）

遠距離に放つ攻撃。  
離れた敵にもダメージを与えられます。

強い敵や危険な状況から  
距離を取って戦うための重要な手段です。

### 🍽 イボ食い（Bキー）

イボを食べて **満腹度を回復**します。

食料が尽きたとき、  
命をつなぐ最後の手段になることも。

---

イボを **攻撃に使うか、生存のために使うか。**

その判断が  
このダンジョンを生き残る鍵になります。
