# ぼじろーぐ (Bojirogue)

ダンジョンの奥深くを目指せ…。

シンプル操作で遊べる、  
絵文字ローグライクダンジョン探索ゲーム。

プレイヤーはダンジョンを降りながら、
モンスターと戦い、アイテムを拾い、成長していきます。  
目指すは最深部―― **30階。**

しかしその先には、  
このダンジョンの主が待っています。

---

## 🎮 Game Features

**シンプル操作 × ローグライク**

- ランダム生成ダンジョン
- 階層ごとに強くなるモンスター
- 食料・武器・装備・特殊アイテム
- 戦略が変わる満腹度システム
- 突然訪れるモンスターハウス
- レベルアップによる成長
- ハイスコアランキング

運が良ければ、  
とてもありがたい存在に出会えるかもしれません。

## 🧭 Goal

最深部 **30階** に到達し、  
ダンジョンの主を倒せ。

ただし――  
帰ってこれるとは限らない。

---

## 🪦 Death is Part of the Game

- 空腹
- 毒
- 強敵
- 不運

このダンジョンでは  
**死ぬこともまた冒険の一部。**

だが、  
次の挑戦ではもっと遠くへ行けるはずだ。
